Состав FarColorer

главная

FarColorer состоит из двух основных частей - непосредственно сам плагин, и базовый набор описаний синтаксисов и стилей раскраски (далее библиотека схем).

Папка bin содержит

  • colorer.dll - библиотека плагина для Far2 x32
  • colorer_x64.dll - библиотека плагина для Far2 x64
  • colorerr.lng - русский языковой файл
  • colorere.lng - английский языковой файл
  • colorerr.hlf - русский файл помощи
  • colorere.hlf - английский файл помощи
  • shortcuts.reg - набор макросов для вызова команд FarColorer из Far2
  • hrcsettings.xml - файл с рекомендуемыми настройками прототипов и типов библиотеки схем
  • colorer.map и colorer_x64.map - map файлы для взаимодействия с FExcept
Папка base содержит библиотеку схем.
